# HR & Talent Lifecycle

This document describes how people enter, move through, and exit the Lucien system.

---

## 1. Talent Sources

- Partners & Experts form
- Manual admin creation
- Converted clients (later)

---

## 2. Talent Profile

A TalentProfile represents a real human.

Key attributes:
- Roles
- Domains
- Seniority
- Availability
- Engagement modes

---

## 3. States

- NEW
- REVIEWED
- APPROVED
- ARCHIVED

State reflects trust and readiness, not employment.

---

## 4. Assignment

Talent can be assigned to:
- Engagements
- Roles within engagements
- With allocation % and visibility flag

Assignments are non-exclusive by default.

---

## 5. Visibility

- Internal-only by default
- Can be shared with client selectively

---

## 6. Exit

Talent is never deleted.
Archived talent remains for audit and history.
