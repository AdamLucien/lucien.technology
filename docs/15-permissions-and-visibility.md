# Permissions & Visibility

Lucien2026 is built for enterprise-grade access control.

---

## 1. Roles

- Anonymous
- Client
- Partner / Talent
- Lucien Operator
- Lucien Admin

---

## 2. Core Principles

- Least privilege
- Explicit sharing
- No implicit cross-org access

---

## 3. Visibility Rules

| Entity | Who can see |
|------|------------|
| Intake Draft | Admin, Operator |
| Inquiry | Admin, Operator, Client |
| Talent Profile | Admin |
| Talent Assignment | Admin, Operator |
| Engagement | Assigned staff + Client |

---

## 4. Client Safety

Clients never see:
- Internal scoring
- Control-room telemetry
- Talent notes

---

## 5. Audit

All sensitive changes emit audit events.
